# JDBC Student Management System

A simple Java console-based application demonstrating JDBC operations with MySQL.

## Features
- Insert Student
- View Students
- Update Student
- Delete Student
- Search by ID
- Search by Name
- Transaction handling

## Technologies Used
- Java
- JDBC
- MySQL

## Concepts Implemented
- DAO Pattern
- PreparedStatement
- Transaction Management (commit & rollback)
- Input Validation

## Database Setup
Run the SQL script in `student-management-database.sql` to create the required table.
